About this Graph

The line graph is very useful for displaying trends. The line graph can be used to plot two different numeric variables, or it can be used to plot one numeric variable with respect to time. This graph is useful when a relation is already known to exist, such as when displaying one variable with respect to time. If there are several values for a particular point on the x-axis, the line graph is most likely not the optimal representation of the data. In that case, the scatterplot may be useful if trying to discover whether a trend exists or not.

Marks and Channels

Marks

Marks include the line.

Channels

Channels include the vertical and horizontal position of points on the line on a common scale. If multiple lines are being shown on one visualization, color may also be used to categorize the lines.
